Beginner Guide: Basic Git and GitHub Commands

Git is a distributed version control system used to track changes in code, while GitHub is a platform that allows developers to host and collaborate on Git repositories.

This guide introduces the most important Git and GitHub commands that every beginner should know.

1. Configure Git

Before using Git, you should configure your username and email.

git config --global user.name "OneTime24"
git config --global user.email "onetimehexa@gmail.com"
